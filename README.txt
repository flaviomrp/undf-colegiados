Sistema de Colegiados da UnDF

Arquivos:
- index.html: página principal do sistema. O GitHub Pages abre este arquivo automaticamente.
- dados_colegiados.csv: base em CSV para conferência/backup.

Como publicar no GitHub Pages:
1. Extraia este ZIP.
2. Envie o index.html e o dados_colegiados.csv para a raiz do repositório.
3. Em Settings > Pages, selecione Branch: main e Folder: /(root).
4. Clique em Save.

Funcionalidades:
- pesquisa por nome, colegiado, processo, temática ou função;
- filtro por tipo e temática;
- ranking de participações;
- consulta rápida por pessoa;
- download dos registros filtrados em CSV.
